Current state of application
The application is in a relatively stable state.
You don't need to worry about inputting wrong values in readings or stations as invalid inputs should be detected and a warning message will be displayed.
At least I hope they will...

Release 1
Working towards release 1, the application looked quite stale.
  • Readings were displayed on the dashboard under each station
  • It was not possible to add or remove readings or stations
But once it was complete, more reading fields were added and the dashboard now showed station list and a button to open each station to view the readings.

Release 2
Release 2 brought two major improvements:
  • Overhaul of user experience by providing icons on the dashboard
  • Member registration was now a thing and each member could have their own list of stations!

Release 3
This was the big release introducing features such as:
  • Algorithms to pick up on weather trends
  • Latest weather cards were moved to display on the dashboard and have changing icons depending on the weather
  • New Settings page to allow members keep their information up to date
